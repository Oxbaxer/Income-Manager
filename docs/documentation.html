Présentation

Income Manager est une application web que vous hébergez vous-même sur votre ordinateur ou serveur. Vos données financières restent entièrement privées — elles ne sont jamais envoyées vers un service externe.

L'application fonctionne dans votre navigateur internet (Chrome, Firefox, Edge...) et s'installe en quelques minutes grâce à Docker.

✅
Vos données vous appartiennent
Toutes les informations sont stockées dans un fichier sur votre machine. Vous pouvez les exporter à tout moment depuis les Paramètres.
2

Prérequis

Avant d'installer Income Manager, vous avez besoin d'un seul logiciel : Docker Desktop.

Installer Docker Desktop

1
Téléchargez Docker Desktop
Rendez-vous sur docker.com/products/docker-desktop et cliquez sur le bouton de téléchargement correspondant à votre système (Windows, Mac ou Linux).
2
Lancez l'installation
Double-cliquez sur le fichier téléchargé et suivez les instructions à l'écran. Sur Windows, acceptez l'activation de WSL 2 si cela vous est demandé.
3
Vérifiez que Docker fonctionne
Après redémarrage, vous devriez voir l'icône Docker (🐋) dans la barre des tâches. L'icône doit être verte ou blanche — cela signifie que Docker est prêt.
⚠️
Sur Windows : WSL 2 requis
Si Windows vous demande d'installer WSL 2, acceptez et suivez le lien fourni. C'est un composant gratuit de Microsoft nécessaire pour faire fonctionner Docker.

Téléchargez les fichiers de l'application

Deux options pour récupérer les fichiers :

Option Comment faire Pour qui
Archive ZIP Téléchargez et décompressez le fichier fourni dans un dossier de votre choix (ex: C:\income-manager) Tout le monde ✅
Git Ouvrez un terminal et tapez : git clone [url-du-dépôt] Développeurs

Installation avec Docker Recommandé

Cette méthode fonctionne sur Windows, Mac et Linux. Elle ne nécessite aucune connaissance en programmation.

Étape 1 — Configurer les secrets de sécurité

Le fichier .env contient les clés de chiffrement de vos données. Il doit être créé avant le démarrage.

1
Copiez le fichier modèle
Dans le dossier de l'application, trouvez le fichier .env.example. Faites-en une copie et nommez-la .env (sans le mot "example").
2
Modifiez les clés secrètes
Ouvrez .env avec le Bloc-notes (clic droit → Ouvrir avec → Bloc-notes). Remplacez les deux valeurs change-me-to-a-random... par des phrases aléatoires de votre choix (minimum 20 caractères).
# Contenu du fichier .env à modifier JWT_SECRET=MonPhraseSecrete2026TresLongueEtComplexe JWT_REFRESH_SECRET=AutrePhraseDifferente2026EgalementLongue PORT=3001 NODE_ENV=production
⚠️
Ne partagez jamais ce fichier .env
Ces clés servent à sécuriser vos sessions. Notez-les quelque part en lieu sûr, mais ne les envoyez pas par email ni ne les publiez sur internet.

Étape 2 — Démarrer l'application

1
Ouvrez un terminal dans le dossier de l'application
Windows : Appuyez sur Shift + clic droit dans le dossier → "Ouvrir la fenêtre PowerShell ici"
Mac : Terminal → faites glisser le dossier dans le terminal
Linux : Clic droit → "Ouvrir dans un terminal"
2
Lancez la commande de démarrage
Tapez la commande ci-dessous et appuyez sur Entrée. Le premier lancement peut prendre 2 à 5 minutes (téléchargement des images).
docker compose up -d
✅
L'application est prête !
Quand la commande se termine, ouvrez votre navigateur et allez sur : http://localhost

Arrêter et relancer l'application

ActionCommande
Démarrerdocker compose up -d
Arrêterdocker compose down
Voir les logsdocker compose logs -f
Mettre à jourdocker compose pull && docker compose up -d
💡
Démarrage automatique avec Windows
Dans Docker Desktop → Settings → General, cochez "Start Docker Desktop when you log in". L'application démarrera automatiquement à chaque ouverture de session.

Premier lancement — Créer votre compte

Lors de votre première connexion, vous devez créer un foyer (espace familial partagé) et un compte administrateur.

1
Ouvrez l'application
Dans votre navigateur, allez sur http://localhost (Docker) ou http://localhost:5173 (développement).
2
Cliquez sur "Créer un foyer"
Dans l'écran de connexion, cliquez sur l'onglet Créer un foyer (à droite de "Connexion").
3
Remplissez le formulaire
Nom du foyer : un nom pour votre espace (ex: "Famille Dupont")
Votre prénom : comment vous souhaitez être identifié
Email : votre adresse email (utilisée pour vous connecter)
Mot de passe : choisissez un mot de passe sécurisé (min. 8 caractères)
4
Cliquez sur "Créer le foyer"
Vous êtes automatiquement connecté et redirigé vers le tableau de bord. Les catégories par défaut (Salaire, Loyer, Courses...) sont créées automatiquement.
ℹ️
Inviter d'autres membres
En tant qu'administrateur, vous pouvez inviter d'autres membres dans votre foyer depuis la page Paramètres. Ils auront accès à toutes les données mais ne pourront pas supprimer de catégories.
⚠️
Conservez vos identifiants
Notez votre email et mot de passe. Il n'y a pas de système de récupération de mot de passe par email dans cette version auto-hébergée.

Gérer vos Revenus

La page Revenus liste toutes vos entrées d'argent. Elle est accessible via le menu de gauche (↑ Revenus).

Ajouter un revenu

1
Cliquez sur "Ajouter un revenu"
Le bouton violet en haut à droite de la page ouvre un formulaire.
2
Remplissez les informations de base
Description : libellé du revenu (ex: "Salaire Janvier")
Montant : en euros (virgule ou point acceptés)
Date : date de réception
Catégorie : sélectionnez dans la liste (Salaire, Loyer perçu, Prime...)
3
(Optionnel) Détails de fiche de paie
Cochez "Fiche de paie" pour saisir le détail de votre bulletin : salaire brut, net, cotisations, primes et nom de l'employeur. Ces informations alimentent le graphique "Revenus cumulés".
4
Cliquez sur "Enregistrer"
Le revenu apparaît dans la liste et les graphiques du tableau de bord se mettent à jour automatiquement.

Modifier ou supprimer un revenu

Chaque ligne de la liste possède deux icônes à droite :

  • ✎ (crayon) — ouvre le formulaire de modification avec les données pré-remplies
  • ✕ (croix) — supprime définitivement l'entrée (une confirmation vous est demandée)

Catégories de revenus

Des catégories par défaut sont créées à l'inscription : Salaire, Loyer perçu, Dons, Dividendes, Freelance, Autre. Vous pouvez en ajouter de nouvelles en cliquant sur le bouton "Catégories de revenus" en haut de page.

Navigation dans la liste

Si vous avez beaucoup de transactions, utilisez les boutons Précédent / Suivant en bas de liste pour naviguer entre les pages (10 entrées par page).

ℹ️
Revenu récurrent ?
Si vous avez un salaire fixe mensuel, utilisez plutôt la page Récurrents pour le saisir une seule fois. Il sera enregistré automatiquement chaque mois.

Transactions Récurrentes

Les transactions récurrentes vous permettent de définir une règle une seule fois pour qu'elle soit enregistrée automatiquement à intervalles réguliers. Idéal pour le loyer, les abonnements, le salaire...

✅
Comment ça fonctionne ?
Chaque heure, l'application vérifie si une règle récurrente est due. Si oui, la transaction est créée automatiquement dans vos revenus ou dépenses, et la prochaine date d'échéance est calculée.

Créer une règle récurrente

1
Cliquez sur "+ Nouvelle règle"
En haut à droite de la page Récurrents.
2
Choisissez le type
Cliquez sur Revenu ou Dépense selon la nature de la transaction. La couleur du formulaire change pour vous indiquer le type sélectionné.
3
Remplissez les détails
Montant : la somme fixe à enregistrer à chaque échéance
Fréquence : Hebdomadaire, Mensuelle ou Annuelle
Catégorie : la catégorie à laquelle rattacher la transaction
Description : le libellé (ex: "Loyer", "Netflix", "Salaire")
Date de début : la première date d'exécution
Date de fin : (optionnel) pour les contrats à durée limitée
4
Cliquez sur "Créer la règle"
La règle apparaît dans la section "Actives". La prochaine date d'exécution est affichée sur chaque ligne.

Gérer les règles existantes

ActionComment faire
Suspendre temporairementCliquez sur le bouton ● (cercle vert) — la règle passe en "Inactive" sans être supprimée
RéactiverCliquez sur le cercle ○ dans la section "Inactives"
ModifierCliquez sur l'icône ✎ (crayon) pour changer le montant, la fréquence, etc.
SupprimerCliquez sur ✕ — attention, les transactions déjà créées sont conservées

Objectifs d'épargne

Les objectifs vous permettent de visualiser la progression vers un projet financier : vacances, apport immobilier, achat d'une voiture, fond d'urgence...

Créer un objectif

1
Cliquez sur "+ Nouvel objectif"
En haut à droite de la page Objectifs.
2
Donnez un nom et un montant cible
Nom : décrivez votre projet (ex: "Vacances Grèce", "Voiture", "Apport appartement")
Montant cible : la somme totale à atteindre
Date cible : (optionnel) quand vous souhaitez atteindre l'objectif. Affiche un compte à rebours en jours.
3
Cliquez sur "Créer l'objectif"
L'objectif apparaît avec une barre de progression à 0%. Vous pouvez en créer autant que vous voulez.

Ajouter une contribution

1
Cliquez sur "+ Contribuer"
Bouton disponible sur chaque carte d'objectif.
2
Saisissez le montant et la date
Vous pouvez aussi ajouter une note (ex: "Bonus de fin d'année", "Vente d'objet").
3
(Optionnel) Rendre la contribution récurrente
Cochez "Contribution récurrente" et choisissez la fréquence. Un virement automatique sera simulé chaque semaine, mois ou année.

Lecture de la barre de progression

Barre violette
Progression normale (moins de 100% atteint). Affiche le montant épargné, le total visé, le pourcentage et le montant restant.
Barre verte + "✓ Objectif atteint !"
Félicitations ! Vous avez atteint ou dépassé votre objectif. La barre devient entièrement verte.

Tableau de synthèse

En haut de la page, 3 indicateurs récapitulent l'ensemble de vos objectifs :

  • Objectifs actifs — nombre total de projets en cours
  • Total épargné — somme de toutes les contributions sur tous les objectifs
  • Progression globale — pourcentage moyen d'atteinte de tous vos objectifs

Prévisions & Simulations

La page Prévisions vous permet de simuler la croissance de votre épargne en tenant compte des intérêts composés. C'est l'outil idéal pour planifier votre retraite, un investissement en bourse ou une assurance-vie.

ℹ️
Qu'est-ce que les intérêts composés ?
Chaque mois, vos intérêts sont ajoutés à votre capital et génèrent eux-mêmes des intérêts. C'est l'effet "boule de neige" de l'épargne. Sur 20 ans à 5%, 500€/mois peuvent générer plus de 85 000€ d'intérêts.

Le simulateur interactif

Le simulateur se calcule en temps réel à mesure que vous modifiez les paramètres. Pas besoin de cliquer sur "Calculer".

ParamètreDescriptionExemple
Capital initial La somme de départ déjà disponible 5 000 €
Versement mensuel La somme que vous ajoutez chaque mois 300 €/mois
Taux annuel (%) Le rendement annuel estimé de votre placement 4 à 7% pour un ETF World
Durée (années) Sur combien d'années simuler 20 ans, 30 ans...

Résultats affichés

💰
Capital final
Le montant total accumulé à la fin de la période, intérêts inclus.
📥
Total versé
La somme que vous aurez effectivement mise de côté (capital initial + tous vos versements).
✨
Intérêts générés
La différence : ce que votre argent a produit grâce aux intérêts composés.

Sauvegarder un scénario

1
Cliquez sur "+ Nouveau scénario" (colonne gauche)
Un formulaire s'ouvre pour nommer votre scénario et saisir les paramètres.
2
Nommez votre scénario et sauvegardez
Ex: "Plan retraite agressif", "Livret A 3%", "ETF World optimiste". Vous pouvez créer plusieurs scénarios et basculer entre eux d'un clic pour comparer les résultats.

Paramètres — Membres, Langue & Export

La page Paramètres est accessible via ⚙ Paramètres dans le menu de gauche.

Gérer les membres du foyer Admin uniquement

En tant qu'administrateur, vous pouvez gérer les comptes de tous les membres de votre foyer.

1
Inviter un nouveau membre
Cliquez sur + Inviter un membre. Renseignez son nom, son email, un mot de passe provisoire et son rôle (Membre ou Administrateur). Cliquez sur Créer le compte.
2
Changer le rôle d'un membre
Cliquez sur le bouton ↑ pour promouvoir un membre en administrateur, ou ↓ pour rétrograder un administrateur en membre simple.
3
Supprimer un membre
Cliquez sur le bouton ✕ à droite du membre concerné, puis confirmez la suppression. Note : vous ne pouvez pas vous supprimer vous-même.
RôleDroits
AdministrateurAccès complet + gestion des membres (inviter, modifier rôle, supprimer)
MembreConsultation et saisie de transactions — ne peut pas gérer les membres
🔒
Protection du dernier administrateur
Il doit toujours rester au moins un administrateur dans le foyer. L'application bloque automatiquement toute action qui supprimerait le dernier admin.

Changer la langue

L'application est disponible en Français 🇫🇷 et en English 🇬🇧. Cliquez simplement sur le bouton de la langue souhaitée. Le changement est instantané et mémorisé pour vos prochaines connexions.

Exporter vos données

Vous pouvez télécharger l'intégralité de vos données dans un fichier JSON à tout moment. Ce fichier contient :

  • Toutes vos transactions de revenus et dépenses
  • Vos catégories personnalisées
  • Vos règles récurrentes
  • Vos objectifs d'épargne et contributions
  • Vos scénarios de prévision
1
Cliquez sur "⬇ Exporter JSON"
Le bouton vert dans la section Export / Import.
2
Sauvegardez le fichier téléchargé
Un fichier income-manager-export-YYYY-MM-DD.json est téléchargé automatiquement dans votre dossier Téléchargements.
⚠️
Sauvegardez régulièrement !
Pensez à exporter vos données régulièrement (une fois par mois) et à conserver le fichier en lieu sûr. En cas de réinstallation, vos données seraient perdues sans cette sauvegarde.

Informations du compte

La section "Compte" affiche votre prénom, votre email et votre rôle (Administrateur ou Membre). Ces informations sont en lecture seule dans cette version.

Questions fréquentes

J'ai oublié mon mot de passe, comment faire ?

L'application n'envoie pas d'email de récupération (elle est auto-hébergée, sans service email). Si vous avez oublié votre mot de passe, un administrateur peut mettre à jour la base de données directement, ou vous pouvez créer un nouveau foyer.

L'application est-elle accessible depuis mon téléphone ?

Oui, l'interface est responsive (adaptée aux mobiles). Depuis votre réseau local, accédez à http://[IP-de-votre-ordinateur]. Pour un accès depuis l'extérieur, configurez un reverse proxy (Nginx, Caddy) ou un VPN.

Les graphiques ne s'affichent pas correctement

Assurez-vous que votre navigateur est à jour. L'application est optimisée pour Chrome, Firefox et Edge dans leurs versions récentes (2023+).

Comment supprimer toutes mes données et recommencer ?

Arrêtez l'application (docker compose down), supprimez le dossier data/, puis relancez (docker compose up -d). L'application démarrera comme une installation neuve.

Puis-je utiliser l'application sur plusieurs ordinateurs ?

Oui, installez l'application sur un seul ordinateur (serveur) et accédez-y depuis d'autres appareils via le réseau local. Tous les membres du foyer utilisent la même instance.
